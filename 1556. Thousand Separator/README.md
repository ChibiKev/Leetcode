# 1556. Thousand Separator
Difficulty: Easy

[Link To Leetcode Problem](https://leetcode.com/problems/thousand-separator/)

## Problem
Given an integer `n`, add a dot (".") as the thousands separator and return it in string format.

## Examples
### Example 1:
```
Input: n = 987
Output: "987"
```
### Example 2:
```
Input: n = 1234
Output: "1.234"
```

## Constraints:
- `0 <= n <= 2^31 - 1`
